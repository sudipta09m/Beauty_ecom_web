Build a mobile-first e-commerce website for selling beauty products.

### Tech Stack:

* Frontend: React.js (with Tailwind CSS)
* Backend: Node.js with Express
* Database:

  * MongoDB: for users, orders, reviews, authentication
  * MySQL: for storing product catalog (name, price, category, image path, stock)

---

### UI/UX Design:

* Style: Soft Luxury
* Colors: Pastel pink, nude, white
* Fonts: Clean, elegant (like Playfair Display + Poppins)
* Mobile-first responsive design (priority)

---

FOLDER STRUCTURE 
package.json: root workspace config. Runs both frontend and backend together.
README.md: setup instructions and project notes.
plan.txt: the original build requirements.
.gitignore: ignores env files, logs, builds, and node_modules.
Frontend

frontend/package.json: frontend dependencies and scripts.
frontend/index.html: the HTML shell Vite serves.
frontend/vite.config.js: Vite config.
frontend/.env.example: frontend env example, mainly API URL.
Frontend src

frontend/src/main.jsx: app entry point. Mounts React and router.
frontend/src/App.jsx: top-level app controller. Handles routes, cart, auth, filters, and page rendering.
frontend/src/index.css: all plain CSS styling for the site.
Frontend components

frontend/src/components/Navbar.jsx: top navigation, hamburger menu, cart button.
frontend/src/components/Footer.jsx: footer section.
frontend/src/components/ProductCard.jsx: reusable product tile.
frontend/src/components/SearchFilters.jsx: search and filters UI.
frontend/src/components/SectionTitle.jsx: reusable section heading.
frontend/src/components/CartDrawer.jsx: slide-out cart panel.
Frontend pages

frontend/src/pages/HomePage.jsx: homepage with hero, categories, trending, offers.
frontend/src/pages/ProductListingPage.jsx: grid listing for products.
frontend/src/pages/ProductDetailPage.jsx: product details, gallery, reviews.
frontend/src/pages/AuthPage.jsx: login/register page.
frontend/src/pages/OrderHistoryPage.jsx: order history page.
frontend/src/pages/FeedbackPage.jsx: feedback form page.
frontend/src/pages/WholesalePage.jsx: wholesale inquiry form.
Frontend data, lib, assets

frontend/src/data/products.js: demo product/category data.
frontend/src/lib/api.js: API helper functions for backend requests.
frontend/src/lib/storage.js: localStorage helpers for cart/auth/orders.
frontend/src/assets: reserved for images/icons if you add local assets later.




Backend

backend/package.json: backend dependencies and scripts.
backend/.env.example: backend env example for JWT, MongoDB, MySQL, and port.
Backend src

backend/src/server.js: starts the API server and database connections.
backend/src/app.js: creates the Express app and mounts routes.
Backend config

backend/src/config/env.js: reads environment variables.
backend/src/config/mongo.js: MongoDB connection setup.
backend/src/config/mysql.js: MySQL connection setup.
Backend controllers

backend/src/controllers/authController.js: register/login logic.
backend/src/controllers/productController.js: product listing, detail, trending, offers.
backend/src/controllers/orderController.js: order placement and order history.
backend/src/controllers/reviewController.js: review list/create logic.
backend/src/controllers/feedbackController.js: feedback and wholesale form handling.
Backend routes

backend/src/routes/authRoutes.js: auth endpoints.
backend/src/routes/productRoutes.js: product endpoints.
backend/src/routes/orderRoutes.js: order endpoints.
backend/src/routes/reviewRoutes.js: review endpoints.
backend/src/routes/feedbackRoutes.js: feedback endpoints.
Backend middleware

backend/src/middleware/auth.js: JWT protection.
backend/src/middleware/error.js: central error handling.
Backend models

backend/src/models/User.js: Mongo user model.
backend/src/models/Order.js: Mongo order model.
backend/src/models/Review.js: Mongo review model.
backend/src/models/Feedback.js: Mongo feedback model.
Backend services, utils, seed

backend/src/services/store.js: in-memory fallback data store when databases are not connected.
backend/src/utils/jwt.js: token creation helper.
backend/src/utils/http.js: small HTTP response helper.
backend/src/seed/schema.sql: MySQL schema and sample product seed data.
